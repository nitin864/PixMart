# PixMart 🛒  (CURRETLY UNDER DEVELOPNMENT)

A full-stack grocery delivery app built with Next.js 15, Auth.js v5, MongoDB & Tailwind CSS. Features role-based access for Admins, Users & Delivery Boys with mobile onboarding.
## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Auth.js v5 (NextAuth)
- MongoDB + Mongoose
- Tailwind CSS v4
- Framer Motion

## Getting Started

```bash
git clone https://github.com/nitin864/PixMart.git
cd PixMart
npm install
npm run dev
```

## Environment Variables

Create a `.env.local` file:

```env
MONGODB_URI=
AUTH_SECRET=
AUTH_GOOGLE_ID=
AUTH_GOOGLE_SECRET=
NEXTAUTH_URL=http://localhost:3000
```
