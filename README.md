PixMart 🛒

Smart Grocery Delivery Platform (Under Development)

PixMart is a modern full-stack grocery delivery application built with the latest web technologies. It provides a smooth and responsive shopping experience while supporting role-based dashboards for Admins, Users, and Delivery Partners.
The platform includes secure authentication, product management, order tracking, and delivery management, making it a complete solution for grocery delivery services.

- Next.js 15 (App Router)
- TypeScript
- Auth.js v5 (NextAuth)
- MongoDB + Mongoose
- Tailwind CSS v4
- Framer Motion

## Getting Started


```bash
git clone https://github.com/nitin864/PixMart.git
cd PixMart
npm install
npm run dev
```

## Environment Variables

Create a `.env.local` file:

```env
MONGODB_URI=
AUTH_SECRET=
AUTH_GOOGLE_ID=
AUTH_GOOGLE_SECRET=
NEXTAUTH_URL=http://localhost:3000
```
